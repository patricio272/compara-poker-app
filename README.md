# compara-poker-app
ComparaOnline Poker WebApp made with Node.js v7.4.0, Sails.js v0.12.13 & @angular/cli v1.2.0

### Installation

```sh
$ npm install -g @angular/cli
$ npm install -g sails
```

### Running Backend

```sh
$ cd backend
$ sails lift
```

### Running Frontend

```sh
$ cd frontend
$ ng serve
```

Once everything is up & running, in your browser, go to http://localhost:3000

Have fun!!
